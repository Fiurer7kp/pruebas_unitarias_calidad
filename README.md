# API para aprendizaje de niños

Autores: Sebastian Coral, Daniela Torres

- Host local: `npm run dev` → `http://localhost:5173/`
- Pruebas: `npm test`
- Build: `npm run build` → `dist/`

## Módulos
- Matemáticas: Descomposición de figuras 3D y fórmula de Euler.
- Ciencias Naturales: Ciclo del agua con partículas y controles.
- Ciencias Sociales: Globo interactivo con marcadores.

## Tecnologías
- React + Vite + TypeScript
- TailwindCSS
- Jest + React Testing Library
